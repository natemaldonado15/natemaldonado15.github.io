# Project Pages for the Jekyll Portfolio

Each project folder contains an `index.md` file with the front matter required by the Reddit/Jekyll template:

```yaml
---
layout: post
title: "Project title"
description: "Short project summary used on the home-page card."
skills:
  - "Skill 1"
main-image: /cover.svg
---
```

## Upload Instructions

1. Download and unzip this package.
2. In your GitHub repository, upload the entire `_projects` folder to the repository root.
3. Confirm the final path looks like `_projects/01-hydroflex-grinding-simulation/index.md`.
4. Commit to `main`.
5. Wait for the GitHub Pages deployment to finish.

## Important Publishing Rules

- Do not add confidential Saint-Gobain customer data, internal test plots, proprietary wheel scans, raw LS-DYNA decks, unpublished thesis figures, or unreleased source code.
- The HydroFlex and employer-project pages are written as high-level portfolio summaries. Keep that discipline when adding images.
- The flowmeter, wing panel, and copper extrusion pages already include public visuals copied from the portfolio deck.
- The HydroFlex, Norton Band, and AM-post-processing pages use clean nonconfidential SVG cover images until approved project figures are available.
